第1章 信息与信息技术

1.1 信息与信息处理

信息的概念

信息（Information）是通过特定媒体承载的消息、情报、指令、数据和信号中包含的可传递、可交换内容，通过文字、数字、符号、图像、声音、视频等方式呈现。

生活中的信息案例
  • 羊城通支付：预存款项、扣费金额、余额等信息
  • 导航定位：当前位置、目的地距离、路线规划等信息
  • 学习强国APP：新闻资讯、图文资源、音视频课程等信息

信息的基本特征

载体依附性

信息必须依附载体存在

可传递性

可通过各种渠道传递

共享性

可被多个接收者利用

时效性

具有一定时间价值

价值性

满足人们某种需要

真伪性

有真实与虚假之分

信息处理的一般过程

1. 信息采集

通过调查、观察、测量、搜索等方式获取所需信息

实例：问卷调查收集数据、相机拍摄照片、网上搜索资料

2. 信息存储

将采集到的信息保存起来，以便后续使用

实例：文件存硬盘、数据存数据库、照片存U盘、纸质文件归档

3. 信息加工

对存储的信息进行整理、分析、计算、编辑等处理，使其更有价值

实例：Excel分析数据、Photoshop处理图片、撰写报告整理资料

4. 信息传输

将加工后的信息传递给需要的人或设备

实例：发送电子邮件、网络传输文件、打电话交流信息

5. 信息发布

将信息公开或传递给特定受众

实例：发布网站文章、展示演示文稿、张贴公告、社交媒体分享

1.3 计算机的基本工作原理

计算机硬件的基本组成

1. 存储器

存储指令和数据，分为内存（RAM、ROM）和外存（硬盘、U盘、光盘）

存储单位：字节（B）、KB、MB、GB、TB，1KB=1024B

2. 运算器

进行算术运算（加减乘除）和逻辑运算（比较、判断）

3. 控制器

计算机的"神经中枢"，指挥和协调各部件工作

4. 输入设备

将信息送入计算机，如键盘、鼠标、触摸屏等

5. 输出设备

将处理结果反馈给用户，如显示器、打印机等

CPU（中央处理器）：运算器和控制器集成在一起，是计算机的核心部件

二进制与程序

二进制基础

计算机内部使用二进制，只有0和1两个数码，逢二进一

二进制优势

  • 易于硬件实现
  • 运算法则简单
  • 逻辑性强

进制转换

  • 二进制→十进制：按位加权求和
  • 十进制→二进制：除2取余法
程序与计算机
  • 程序：控制计算机工作的一系列命令集合
  • 机器语言：计算机能直接识别的二进制代码
  • 高级语言：接近自然语言，需编译转换（如Visual Basic、C语言）
  • 计算思维：像计算机科学家一样思考问题的方式，是核心素养

自我复制、破坏系统

3.2 图像的采集与简单加工

图像的采集方法

数码相机/手机拍摄

直接获取数字图像，通过数据线或读卡器导入电脑

扫描仪扫描

将纸质图像、实物转换为数字图像

屏幕截图

按PrintScreen键或使用截图工具（如QQ截图）

网上下载

从图库网站、搜索引擎获取图像（注意版权）

图像数字化原理

图像通过像素点记录颜色信息，分为位图和矢量图：

位图（像素图）

由像素组成，放大失真，适合照片、复杂图像

格式：JPG、PNG、BMP、GIF

矢量图

由数学指令描述，放大不失真，适合标志、插图

格式：AI、SVG、CDR

图像文件格式与加工

常见图像格式特点
格式 特点 适用场景
JPG/JPEG 有损压缩，文件小，画质好 照片、复杂图像
PNG 无损压缩，支持透明背景 图标、Logo、透明图像
GIF 支持动画，最多256色 简单动画、图标
BMP 无压缩，文件大，画质高 高质量图像存储
图像加工操作（以Photoshop为例）
  • 裁剪：去除多余部分，调整构图（快捷键C）
  • 调整大小：修改图像分辨率和尺寸（图像→图像大小）
  • 亮度/对比度调整：改善图像明暗（图像→调整→亮度/对比度）
  • 色彩调整：修改色相、饱和度（图像→调整→色相/饱和度）
  • 去除水印：使用仿制图章工具、修补工具
  • 添加文字：使用文字工具添加标题和说明

3.3 声音的采集与简单加工

声音的采集方法

计算机录音

使用Windows录音机或Audacity，配合麦克风

默认录音60秒，可延长或减速增加时间

CD音频提取

使用Windows Media Player复制CD曲目，保存为WMA格式

网上下载

从音乐网站（如百度MP3）下载音频文件

专业设备录音

使用录音笔、智能手机录音，再导入电脑

声音数字化原理

声音通过"采样→量化→编码"转换为数字信号：

  1. 采样：按固定时间间隔采集声音信号
  2. 量化：将采样值转换为数字量
  3. 编码：将数字量转换为二进制代码

音频文件格式与加工

常见音频格式特点
格式 特点 适用场景
WAV 无压缩，音质好，文件大 高质量音频存储
MP3 有损压缩，文件小，音质较好 音乐播放、网络传输
WMA 压缩比高，音质优于MP3 Windows平台音频
MIDI 记录音乐指令，文件小，非音频波形 电子音乐、游戏音效
音频加工操作（以Cool Edit Pro为例）
  • 音量调整：效果→波形振幅→音量标准化
  • 截取片段：选中区域→文件→保存选取区域
  • 删除片段：选中区域→编辑→删除选取区域
  • 降噪处理：选中噪音样本→效果→噪音消除→降噪器
  • 音效添加：添加回声、混响、合唱等效果
  • 格式转换：保存时选择目标格式（如MP3）

3.4 视频的采集与简单加工

视频的采集方法

实地拍摄

使用数码摄像机（DV）、智能手机拍摄

适合记录活动、场景（如校运会、文艺汇演）

网上下载

从视频网站（如优酷、腾讯视频）下载

注意版权，选择合适格式（如MP4、FLV）

屏幕录制

使用屏幕录制软件（如Camtasia）录制屏幕操作

适合制作教程、演示视频

设备导入

通过数据线将摄像机、手机中的视频导入电脑

或使用读卡器读取存储卡

视频文件格式
普通视频格式
  • AVI：早期格式，文件大
  • MP4：压缩比高，兼容性好
  • MOV：iOS平台格式
  • MKV：高清格式，支持多音轨
流媒体格式
  • FLV/F4V：网页视频常用
  • RM/RMVB：RealPlayer格式
  • WMV：Windows流媒体
  • 3GP：手机视频格式

视频的加工与格式转换

视频加工操作（以QQ影音为例）
  • 截取视频片段：
    1. 打开视频，右键→工具→截取
    2. 设置起始时间和结束时间
    3. 选择输出格式和分辨率
    4. 点击"保存"完成截取
  • 视频合并：使用专业软件（如会声会影）将多个视频片段拼接
  • 添加字幕：导入字幕文件（如SRT格式），调整显示位置和时间
  • 添加转场效果：在片段之间添加过渡动画（如淡入淡出、旋转）
视频格式转换（以格式工厂为例）
  1. 打开格式工厂，选择"视频"→"所有转到MP4"（或其他目标格式）
  2. 点击"添加文件"，选择需要转换的视频
  3. 设置输出文件夹和参数（如分辨率、比特率）
  4. 点击"开始"，等待转换完成

常见转换需求

FLV→WMV（用于插入演示文稿）、AVI→MP4（减小文件体积）

3.5 多媒体作品的集成与评价

多媒体作品集成（以WPS演示为例）

集成步骤
  1. 整理素材：

    按文本、图像、音频、视频分类存放，建立文件夹结构

  2. 制作封面和目录：

    设计封面背景，添加标题；制作目录页，设置导航链接

  3. 添加内容素材：

    插入文本（文本框）、图像（插入→图片）、音频（插入→媒体剪辑→声音）、视频（插入→视频）

  4. 设置动态效果：

    添加幻灯片切换效果（动画→切换到此幻灯片）和对象动画（动画→自定义动画）

  5. 添加导航链接：

    插入超链接，实现页面跳转、外部文件链接（插入→超链接）

集成注意事项
  • 素材路径：确保音频、视频文件与演示文稿在同一文件夹，避免链接失效
  • 风格统一：保持字体、颜色、版式风格一致，提升作品美观度
  • 交互设计：合理设置导航，方便用户操作（如目录链接、返回按钮）
  • 兼容性：选择通用格式，确保在不同设备上正常播放

多媒体作品的测试与评价

作品测试
  • 内容测试：检查内容完整性、准确性，有无错别字
  • 功能测试：测试动画效果、超链接、音视频播放是否正常
  • 兼容性测试：在不同计算机、不同软件版本中测试
  • 性能测试：检查文件大小、加载速度，优化冗余内容
作品打包

WPS演示：文件→文件打包→将演示文稿打包成文件夹，便于在其他电脑播放

作品评价标准
评价维度 评分标准（总分100分）
主题与思想性（30分） 主题鲜明，内容充实，有思想性（20-30分）；有主题但缺乏深度（10-19分）；主题模糊（10分以下）
技术与科学性（40分） 技术应用恰当，无科学性错误（30-40分）；技术使用合理（20-29分）；技术应用不足（10-19分）；有错误（10分以下）
艺术性（30分） 画面美观，风格统一（20-30分）；效果一般（10-19分）；排版混乱（10分以下）

优化建议

根据评价反馈，调整内容结构、优化视觉效果、修复功能问题
