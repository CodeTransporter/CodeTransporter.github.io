二、选择题

1. 在WPS表格中，若要固定某列数据不随滚动条移动，应使用的功能是（ ）

A. 合并单元格
B. 冻结窗格
C. 筛选
D. 分类汇总
显示答案

答案：B

解析：冻结窗格功能可固定指定的行或列，方便查看大量数据时对照表头。

2. 下列关于单元格引用的说法，正确的是（ ）

A. 相对引用（如A1）复制公式时地址不变
B. 绝对引用（如$A$1）复制公式时地址随位置调整
C. 混合引用（如$A1或A$1）可固定行或列
D. 单元格引用只能引用同一工作表中的单元格
显示答案

答案：C

解析：相对引用复制时地址自动调整，绝对引用地址固定，混合引用固定行或列；引用可跨工作表（如Sheet2!A1）。

3. 对表格数据进行分类汇总前，必须先进行的操作是（ ）

A. 筛选数据
B. 排序数据
C. 合并单元格
D. 插入图表
显示答案

答案：B

解析：分类汇总需先按分类字段排序，确保相同类别的数据集中在一起，否则汇总结果会分散。

3. 编写程序：使用while循环打印9×9乘法表（格式如下）：

1×1=1
1×2=2 2×2=4
1×3=3 2×3=6 3×3=9
...
1×9=9 2×9=18 ... 9×9=81
显示参考代码 
# 9×9乘法表（while循环）
i = 1  # 控制行数
while i <= 9:
    j = 1  # 控制每行的列数
    while j <= i:
        # end=" "使不换行，格式化输出对齐
        print(f"{j}×{i}={j*i}", end="\t")
        j += 1
    print()  # 换行
    i += 1

4. 编写程序：输入5名学生的语文成绩，计算并输出以下信息：

  • 5名学生的成绩列表
  • 总成绩和平均成绩
  • 最高成绩和最低成绩
  • 成绩≥80分的学生人数
显示参考代码 
# 学生成绩统计
scores = []  # 存储成绩的列表
count_80 = 0  # 统计≥80分的人数

# 输入5名学生成绩
for i in range(5):
    score = int(input(f"请输入第{i+1}名学生的语文成绩："))
    scores.append(score)
    if score >= 80:
        count_80 += 1

# 计算统计信息
total = sum(scores)
average = total / 5
max_score = max(scores)
min_score = min(scores)

# 输出结果
print("\n5名学生的成绩列表：", scores)
print("总成绩：", total)
print("平均成绩：", round(average, 1))  # 保留1位小数
print("最高成绩：", max_score)
print("最低成绩：", min_score)
print("成绩≥80分的人数：", count_80)
