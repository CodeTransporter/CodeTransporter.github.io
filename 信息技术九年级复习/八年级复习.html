第1章 数据的分析与处理

1.1 认识WPS表格

表格界面组成

  • 标题栏：显示文件名和软件名称
  • 功能区：包含所有操作工具，按用途分类在选项卡中
  • 编辑区：表格主体区域，由行和列组成
  • 名称框：显示当前选中单元格或区域的名称
  • 编辑栏：输入和编辑单元格内容
  • 状态栏：显示表格状态信息（如选中区域统计）
文件格式

WPS表格默认格式为.et，可另存为.xlsx格式以便兼容Excel

表格核心元素

工作簿

一个WPS表格文档就是一个工作簿，扩展名为.et或.xlsx，可包含多个工作表

工作表

工作簿中的单个表格页面，默认包含Sheet1、Sheet2、Sheet3，可重命名和增减

行和列

行以数字标识（1、2、3...），列以字母标识（A、B、C...AA、AB...）

单元格和区域

行与列的交汇处，名称由列标+行号组成（如A1、B3）；多个单元格组成区域（如A1:C5）

1.2 表格数据获取

基础数据输入

文本输入

选中单元格直接输入，默认左对齐，支持汉字、字母、数字和符号组合

数值输入

先设置单元格格式（数值、货币、百分比等），再输入数值，默认右对齐

注意：WPS表格有效数值为15位，第15位后数字将转为0

日期和时间输入

日期用"/"或"-"分隔（如2024/05/20），时间用":"分隔（如14:30:00）

可通过单元格格式设置显示样式（如2024年5月20日）

高效数据输入

有规律数据填充
  1. 在单元格中输入起始数据（如A001）
  2. 鼠标移至单元格右下角，变为黑色十字（填充柄）
  3. 按住左键拖动至目标单元格
  4. 选择"以序列方式填充"完成自动填充
外部数据获取
  • 从WPS文档复制表格数据，直接粘贴到表格中
  • 导入CSV、TXT等格式的外部数据文件
  • 通过"数据"选项卡中的"自文本"功能导入数据

1.3 表格数据处理

表格样式与排序

表格样式设置

选中表格区域 → 开始选项卡 → 表格样式 → 选择合适样式，可实现隔行填充、自动汇总等效果

数据排序
  • 单关键字排序：选中数据区域 → 数据选项卡 → 排序 → 选择关键字和排序方式（升序/降序）
  • 多关键字排序：添加次要关键字，解决主要关键字相同的排序问题
  • 排序规则：数字按大小、日期按先后、文本按字母顺序

数据筛选与分类汇总

数据筛选
  1. 选中数据区域任意单元格
  2. 数据选项卡 → 自动筛选
  3. 点击表头下拉按钮 → 选择筛选条件（如文本筛选、数值筛选）
  4. 支持多条件筛选和二次筛选
分类汇总
  1. 先对分类字段进行排序（如按"品种"排序）
  2. 数据选项卡 → 分类汇总
  3. 设置分类字段、汇总方式（求和/平均值/计数等）、汇总项
  4. 通过左侧+/-按钮展开/折叠汇总结果

1.4 表格数据计算

公式与运算符

公式输入

以等号"="开头，后跟表达式（如=D2+E2+F2+G2），按回车计算结果

支持复制公式：选中含公式单元格 → 复制 → 粘贴到目标单元格，公式会自动调整引用

常用运算符

算术运算符

+（加）、-（减）、*（乘）、/（除）、^（乘方）、%（百分比）

比较运算符

=（等于）、>（大于）、<（小于）、>=（大于等于）、<=（小于等于）、<>（不等于）

引用运算符

:（区域）、,（联合）、空格（交叉）

文本运算符

&（连接文本，如"A"&"B"="AB"）

单元格引用与函数

单元格引用

相对引用

如D2，复制公式时引用地址随位置自动调整

绝对引用

如$D$2，复制公式时引用地址固定不变

常用函数

SUM(范围) - 求和

例：=SUM(D2:G2) 计算D2到G2的总和

AVERAGE(范围) - 求平均值

例：=AVERAGE(D2:D10) 计算D列数据的平均值

MAX(范围) / MIN(范围) - 求最大/最小值

例：=MAX(D2:D10) 找出D列最大数值

COUNT(范围) - 计数

例：=COUNT(D2:D10) 统计D列数值型数据的个数

1.5 图表制作

图表基础

图表组成要素
  • 图表标题：说明图表主题
  • 坐标轴：X轴（分类轴）、Y轴（数值轴）
  • 数据系列：图表中代表数据的图形（如柱形、折线）
  • 图例：解释数据系列的含义
  • 网格线：辅助读取数据数值
常用图表类型及适用场景

柱形图

比较不同类别数据的差异，展示一段时间内的数据变化

条形图

适用于分类标签较长的情况，便于横向比较

折线图

强调数据的变化趋势和时间关联性

饼图

展示各部分占总体的比例关系

图表创建与美化

创建图表步骤
  1. 选中要创建图表的数据区域（包括表头）
  2. 插入选项卡 → 图表 → 选择图表类型（如柱形图）
  3. 系统自动生成图表，可拖动图表调整位置和大小
  4. 通过"图表工具"选项卡编辑图表元素
图表编辑与美化
  • 编辑图表标题：点击标题直接修改，或通过"图表元素"添加
  • 更改图表类型：图表工具 → 更改类型 → 选择新类型
  • 调整数据来源：图表工具 → 选择数据 → 修改数据区域
  • 美化设置：设置图表颜色、填充效果、边框样式，添加数据标签
  • 添加趋势线：用于显示数据变化趋势，辅助数据分析

第2章 程序设计初步

2.1 程序设计语言与流程图基础

程序与程序设计语言

计算机程序

控制计算机工作的一系列命令集合，按照特定逻辑组织，让计算机完成指定任务

程序设计语言分类

机器语言

计算机能直接识别的二进制代码，执行效率最高但难以编写

汇编语言

用助记符代替机器指令，需要汇编器转换为机器语言

高级语言

接近自然语言和数学表达式，易读易写，需编译或解释执行（如Python、C、Java）

算法与流程图

算法

解决问题的步骤和方法，必须满足有限性、可行性、确定性要求，可用自然语言、伪代码或流程图描述

流程图符号及规则
符号 名称 功能
开始/结束 表示程序的开始或结束
输入/输出 表示数据的输入或结果的输出
处理过程 表示数据处理或运算操作
判断 表示条件判断，有两个分支
流程线 表示程序执行的方向

流程图规则：从上到下、从左到右绘制，只有一个起点，至少一个终点

2.2 Python编程基础

数据类型与变量

基本数据类型

整数（int）

如10、-5、0，支持任意大小整数

浮点数（float）

如3.14、-0.5、1.23e5（科学记数法）

字符串（str）

用单引号、双引号或三引号括起来的文本，如"Hello"、'Python'

布尔值（bool）

只有True（真）和False（假）两种值，用于逻辑判断

空值（None）

表示无值，与0和空字符串不同

变量命名规则
  • 由字母、数字、汉字和下划线组成
  • 不能以数字开头
  • 不能包含空格
  • 区分大小写（如name和Name是不同变量）
  • 不能使用Python关键字（如if、for、while等）

输入输出与语句

输入函数input()

获取用户键盘输入，返回字符串类型

格式：变量 = input("提示文字")

示例：name = input("请输入你的名字：")

注意：输入数值需转换类型，如age = int(input("请输入年龄："))

输出函数print()

在屏幕上显示输出内容

格式1：print(表达式) → 输出单个内容

格式2：print(表达式1, 表达式2, ...) → 输出多个内容，用空格分隔

示例：print("Hello,", name)、print("1+2=", 1+2)

注释语句
  • 单行注释：以#开头，#后的内容不执行
  • 多行注释：用三个单引号'''或三个双引号"""括起来
  • 作用：解释代码功能，增强程序可读性

2.3 分支结构——让机器学会做选择

条件表达式

关系运算符

==

等于

!=

不等于

>

大于

<

小于

>=

大于等于

<=

小于等于

逻辑运算符

and

与运算

两边都为真才为真

or

或运算

至少一边为真即为真

not

非运算

取反（真变假，假变真）

条件语句格式

单选结构（if语句）

格式：

if 条件表达式:
    语句块  # 缩进（Tab键）

功能：条件为True时执行语句块，否则跳过

示例：判断PM2.5指数大于200时提醒戴口罩

二选一结构（if-else语句）

格式：

if 条件表达式:
    语句块1  # 条件为真时执行
else:
    语句块2  # 条件为假时执行

功能：二选一执行，必执行其中一个语句块

示例：判断成绩是否及格（>=60分为及格）

多选一结构（if-elif-else语句）

格式：

if 条件1:
    语句块1
elif 条件2:
    语句块2
...
else:
    语句块n

功能：多个条件依次判断，只执行第一个满足条件的语句块

示例：成绩等级评定（优秀、良好、合格、不合格）

2.4 循环结构——让机器重复工作

for循环

基本格式

格式：

for 循环变量 in range(参数):
    语句块  # 缩进

range()函数用法：

  • range(n)：生成0~n-1的整数序列（如range(3) → 0,1,2）
  • range(a,b)：生成a~b-1的整数序列（如range(1,4) → 1,2,3）
  • range(a,b,c)：生成a~b-1、步长为c的序列（如range(1,10,2) → 1,3,5,7,9）
应用示例

计算1+2+3+...+n的和：

n = int(input("请输入n的值："))
sum = 0
for i in range(1, n+1):
    sum += i  # 等价于sum = sum + i
print("总和 =", sum)

while循环与break语句

while循环基本格式

格式：

while 条件表达式:
    语句块  # 缩进
    循环变量更新  # 避免无限循环

功能：条件为True时重复执行语句块，直到条件为False结束

示例：用while循环计算1+2+...+n的和

n = int(input("请输入n的值："))
sum = 0
i = 1
while i <= n:
    sum += i
    i += 1  # 循环变量自增
print("总和 =", sum)
break语句

用于跳出循环，执行循环后的语句

应用场景：无限循环中根据条件退出

# 猜数字游戏示例
target = 88
while True:
    guess = int(input("请猜一个数字(1~100)："))
    if guess == target:
        print("猜对了！")
        break  # 跳出循环
    elif guess > target:
        print("猜大了！")
    else:
        print("猜小了！")

2.5 Python库——让强大的Python库帮忙

库的导入与使用

库的导入方式

import 库名

导入整个库，使用格式：库名.函数名()（如import random）

from 库名 import 函数名

导入库中的指定函数，直接使用函数名()（如from random import randint）

import 库名 as 别名

给库起别名，使用更简洁（如import turtle as t）

random库（随机数库）

random.randint(a,b)

生成a~b之间的随机整数（包含a和b）

random.random()

生成0~1之间的随机小数（不包含1）

random.choice(列表)

从列表中随机选择一个元素

应用：猜数字游戏中随机生成目标数字

turtle库（绘图库）

常用绘图函数

forward(d)

向前移动d像素

backward(d)

向后移动d像素

right(angle)

向右转angle度

left(angle)

向左转angle度

circle(r)

绘制半径为r的圆

pencolor(color)

设置画笔颜色

绘图示例：绘制正方形
import turtle
t = turtle.Turtle()
for _ in range(4):
    t.forward(100)  # 向前移动100像素
    t.right(90)     # 向右转90度
turtle.done()  # 保持绘图窗口

第3章 App Inventor手机积木式编程

3.1 认识App Inventor

App Inventor概述

基于Web的Android应用开发工具，无需编写代码，通过拖拽拼接积木模块即可制作手机APP，适合编程初学者

支持开发多媒体应用、游戏应用、位置感知应用、基于Web的应用等

启动方式
  • 离线版：安装本地服务器，启动服务后通过浏览器访问（如192.168.0.11:8888）
  • 在线版：通过广州智慧教育公共服务平台的APP INVENTOR栏目，注册账号后在线开发

开发界面组成

设计界面
  • 组件面板：提供各类可拖拽的组件（用户界面、布局、多媒体等）
  • 工作区域：可视化设计APP界面，摆放组件的画布
  • 组件列表：显示当前页面已添加的组件，可重命名
  • 属性面板：设置选中组件的属性（如标题、颜色、大小等）
  • 素材面板：管理APP所需的图片、音频等素材
编程界面
  • 代码块面板：按组件和功能分类的积木模块库
  • 工作区域：拼接积木模块，设计APP逻辑功能
  • 核心概念：事件（触发动作）、方法（组件功能）、属性（组件特征）
